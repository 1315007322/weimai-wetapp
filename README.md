# weimai-wetapp
微麦电影购票小程序，分为三部分：小程序端、商家（影院）后台与官方后台
> 还有很大的优化空间，看几时有空

## 目录说明
- **weapp-weimai**：小程序页面
- **film_admin**：管理后台Web页面
- **weimai**：整个项目的后端

## 相关技术
- **前端**：Vue、ElementUI...
- **后端**：SpringBoot、MyBatis、Druid、MySQL、Shiro、Elasticsearch、Redis...
